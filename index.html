Our Mission

Our goal with this project was to create a mini dictionary that is also fun! It is our belief that having context and a visual aid paired with the definition will not only make the definition stick better but will make expanding ones vocabulary a fun experience.

APIs

This project was made with the Oxford Dictionary API and the Giphy API to create this project.
